# アプリ概要
ホテルを予約サイトであり、管理者側がサイトのデザインなどを調整できて手軽に使えます。
スライドバーの画像変更や記事の投稿など他にもあります。

# アプリ開発の背景
年に数回旅行する際にホテルの予約をするときに私は泊まるホテルのサイトを見て決めているのでその際にホテルのサイトにはたくさんの機能があり、
自らのアウトプットの機会になると思い制作を決めました。
